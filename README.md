# investment-portfolio

Проект по управлению инвестиционным портфелем для российских акций.

## Описание

Проект реализует комплексный анализ портфельных стратегий на основе данных о ценах 30 российских компаний с Московской биржи (IMOEX) за период 2015-2025 гг.

### Основные задачи

1. **Задачи 2-3**: Векторы доходностей и ковариационные матрицы
   - Расчет логарифмических доходностей
   - Анализ с использованием скользящего окна (1 год)
   - Анализ с использованием расширяющегося окна
   - Экспоненциальное забывание (λ=0.94)

2. **Задачи 9-10**: Анализ стабильности во времени границы эффективных портфелей
   - Построение эффективных границ для разных исторических окон
   - Сравнение методов: скользящее окно, расширяющееся окно, экспоненциальное забывание
   - Анализ стабильности портфельных весов
   - Вычисление метрик эффективности (Шарп-отношение, коэффициент эффективности)

3. **Задачи 11-12**: Сравнение различных подходов к оценке входящих данных
   - Выбор индекса для расчета β-коэффициентов (IMOEX)
   - Обоснование выбора исторического окна (1 год)
   - Обоснование схемы взвешивания наблюдений (равные веса)

4. **Задачи 13-15**: Оценка входящих данных на основе исторических β
   - Расчет β-коэффициентов на основе рыночной модели (r_i = α_i + β_i × r_m + ε_i)
   - Расчет ковариационной матрицы из β (с учетом остаточных дисперсий)
   - Построение эффективной границы на основе β-модели
   - Анализ динамики эффективных границ на основе β во времени

## Метрики и результаты

### Ключевые метрики

1. **Доходность (Return)**: Среднее изменение цены за период (логарифмическая)
2. **Риск (Risk)**: Стандартное отклонение доходности
3. **Шарп-отношение (Sharpe Ratio)**: Доходность / Риск (без безрисковой ставки)
4. **Коэффициент эффективности**: Максимальная доходность / Минимальный риск
5. **GMVP (Global Minimum Variance Portfolio)**: Портфель с минимальным риском
