## Теоретическая основа

### Задача 22: Two-Fund Theorem

Согласно теореме Блэка (1972), если $P_1$ и $P_2$ — любые портфели на эффективной границе, то любой третий портфель $P_3$ на этой же границе может быть представлен как:

$$P_3 = \lambda P_1 + (1 - \lambda) P_2$$

для некоторого $\lambda \in [0, 1]$.

### Задача 23: Монте-Карло

Эффективная граница через случайный поиск:

1. Генерировать случайные веса $w_i^{(k)}$ такие что $\sum w_i = 1$
2. Рассчитать $r_p^{(k)} = \sum w_i^{(k)} \bar{r}_i$
3. Рассчитать $\sigma_p^{2(k)} = w^{(k)} \Sigma (w^{(k)})^T$
4. Портфель A доминирует B если $r_A \geq r_B$ и $\sigma_A \leq \sigma_B$
5. Оставить только недоминируемые портфели

### Задача 24: Наиболее рискованный портфель

Максимизация дисперсии вместо минимизации:

$$\max_{w} w^T \Sigma w$$

при условии $\sum w_i = 1$

Решение через собственные векторы: использовать собственный вектор с максимальным собственным значением.

### Задача 25: Implementation Shortfall

IS — это ожидаемое значение отрицательных отклонений от бенчмарка:

$$IS = E[\min(R - B, 0)] = \frac{1}{N} \sum_{i=1}^{N} \max(B - R_i, 0)$$

Преимущества:
- Учитывает асимметрию распределения
- Фокусируется на убытках (убытки vs прибыль)
- Более робастен к выбросам, чем дисперсия
